# Grill & Chill - Restaurant Website

A modern, responsive website for your restaurant "Grill & Chill" featuring a menu, about section, and contact form.

## Features

✨ **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
🎨 **Modern UI** - Clean and professional design with smooth animations
📱 **Mobile-First** - Optimized for all screen sizes
⚡ **Fast Loading** - Lightweight HTML, CSS, and JavaScript
🔄 **Smooth Navigation** - Smooth scroll transitions between sections
🎯 **Call-to-Action** - Menu and contact sections encourage customer engagement

## Project Structure

```
grill-and-chill/
├── index.html      # Main HTML file with all page content
├── styles.css      # All styling and responsive design
├── script.js       # Interactive features and animations
└── README.md       # This file
```

### index.html

### script.js
- Smooth scroll navigation
- Active link highlighting during scroll

## How to Use

1. **Open in Browser**: Simply open `index.html` in your web browser
2. **No Installation Needed**: This is a static website with no dependencies
3. **Local Development**: Use VS Code with Live Server extension for auto-refresh

## Customization

### Change Restaurant Name
Replace "Grill & Chill" throughout the files with your restaurant name.

### Update Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #d4443a;      /* Main red color */
    --secondary-color: #2c3e50;    /* Dark blue color */
    --light-bg: #f8f9fa;           /* Light background */
}
```

### Add Your Menu Items
Edit the menu section in `index.html` by adding/removing `.menu-item` divs

### Update Contact Information
Replace placeholder information in the contact section with your actual details

### Add Images
Create an `images/` folder and replace emoji placeholders with actual images
